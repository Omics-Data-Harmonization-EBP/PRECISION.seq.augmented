# Create precision object for clustering

Use this function to create a
[`precision`](https://omics-data-harmonization-ebp.github.io/PRECISION.seq.augmented/reference/precision-class.md)
object for clustering tasks. This object will contain raw training data
and its labels.

## Usage

``` r
create.precision.cluster(data, label)
```

## Arguments

- data:

  Data matrix

- label:

  Labels

## Value

A precision object for clustering
